To setup this project in your local:

1. Download node.js from http://nodejs.org/

2. Follow the instructions to install

3. Download and install the appropriate version of MongoDB for your operating system from http://www.mongodb.org/downloads

4. Download the NodeEclipse IDE Enide-Studio-2014-011-20140228-win64 from http://sourceforge.net/projects/nodeclipse/files/Enide-Studio-2014/0.11-preview/. You can chose any other IDE also.

5. Go to the folder and run npm install. This will download the necessary node js module in your local env.

6. Run the project node app.js (from the command line) or directly run it from the IDE.
